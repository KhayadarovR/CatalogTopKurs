# CatalogTopKurs

Использована (стэк):
- MVC
- AccountService: вход\регистрация, claim в куки, хеширование пароля
- EF core, DB first подход (PostgreSQL)
- DI контейнеры, внедрение зависемостей
- Bootstrap 5
